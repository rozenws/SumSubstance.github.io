          # WebSDK customization

          WebSDK is flexible from the point of view of customization, you can add or change almost any text, use css, so that the webSDK was in the style of your service design.

          You can play with customization here.

          # init

          Name Description
          includedCountries List of included countries
          excludedCountries List of excluded countries

          # navConf

          Name Description
          skipWelcomeScreen Skip an additional screen which contains button Upload new documents
          showWarningScreen Display an additional screen with attention for the user with company info
          skipReviewScreen Skip the Review screen, the applicant will switch to pending status
          skipAgreementsScreen Skip Agreement screen with general terms and conditions

          # uiConf

          Name Default Description
          customCss Ability to use your CSS-file with customized settings
          lang en Language which shows webSDK
          documentExamples You can change the document examples

          Changing lang at uiConf of your WebSDK initialization also changes lang parameter for an applicant to provide moderation comments for your users in this language.

          # steps

          Name Description
          instructions Changes the text on the screen
          title Changes the title on the screen
          videoRequired Using camera for taking photo/video
          customFields Adding other fields if needed

          You can play with customization here.

          Initialize a WebSDK (no effort):

          Copy
          <script>
  var id = idensic.init(
    '#idensic',
    {
           "clientId": "YourClientId",
           "externalUserId": "someUserIdInYourSystem",
           "accessToken": "_act-168c2e88-8fed-4723-8bcd-73d4d0aff7a6",
           "navConf": {
             "showWarningScreen": true
           },
           "uiConf": {
             "customCss": "",
             "documentExamples": {
               "SELFIE":['YOUR_URL','YOUR_URL','YOUR_URL']
              },
             "steps": {
               "CREATE_APPLICANT": {
                 "instructions": "To get your account verified, please prepare your identity documents – Passport, National ID card or Driving license. Please make sure that your document is valid. To get detailed information on how to pass KYC, please read the complete guide.\n\n**Note**. Do not try to use fake documents. It will lead to the final blocking of the account.",
                 "title": "Please follow the steps after agreeing to our conditions.",
                 "agreements": [
                   "By clicking the “I Agree” button I accept [General Terms and Conditions](https://sumsub.ru/privacy-policy/)",
                   "I give my consent to process my personal data for the purposes and conditions set out in the Privacy Policy."
                 ]
               },
               "IDENTITY": {
                 "title": "Custom title for the identity step"
               },
               "SELFIE": {
                 "videoRequired": "liveness"
               },
             }
           }
        })
</script>

          # Supported documents by countries

          To display different set of documents for different countries you can use documentsByCountries structure.

          Note: this setting reflects only frontend part of the WebSDK and applicant's docSets stays the same as it was set by applicant request or via creation of the applicant.

          Copy
          <script>
  var id = idensic.init(
    '#idensic',
           "clientId": "YourClientId",
           "externalUserId": "random-qofbcxlwuy24554793",
           "accessToken": "_act-c9750532-523b-44d4-8f5d-cb0c996f22f4",
           "navConf": {
             "showWarningScreen": true,
             "skipWelcomeScreen": false,
             "skipReviewScreen": true,
             "skipAgreementsScreen": false
           },
           "documentsByCountries": {
             "OTHER": {
               "PASSPORT": {
                 "doubleSided": false
               },
               "ID_CARD": {
                 "doubleSided": true
               },
               "RESIDENCE_PERMIT": {
                 "doubleSided": true
               }
             },
             "USA": {
               "PASSPORT": {
                 "doubleSided": false
               },
               "DRIVERS": {
                 "doubleSided": true
               },
               "ID_CARD": {
                 "doubleSided": true
               }
             },
             "CAN": {
               "PASSPORT": {
                 "doubleSided": false
               },
               "ID_CARD": {
                 "doubleSided": true
               },
               "DRIVERS": {
                 "doubleSided": true
               }
             }
           }
        })
</script>
