# Reusable KYC

This option is available to any client who uses WebSDK.

The process looks pretty simple:

  • We check whether a user already passed verification with the same email and got approved for some client.
  • We ask a user whether they want to confirm that.
  • If they agree, they pass a liveness check, and we compare their face with the ID documents.
  • If all is okay, we just copy their documents to your applicant.

If you want to use this option, get in touch with the Sum&Substance team.
