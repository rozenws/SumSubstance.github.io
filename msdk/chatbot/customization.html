      # Customization

      You can customize some flow settings and localization in our dashboard. Go to test dashboard or production dashboard to do so.

      # Flow customization

      At the moment you can customize the following things:

      • Skip agreements screen (in case you have your own in your app)
      • Require two pages for passports
      • Disable liveness check remotely (in case for some reason you want record just a video)
      • Show a close button within our chat when all documents were submitted by a user

      # Localization customization

      Localization customization is done via a JSON file that is shown in our dashboard. By default a JSON file is returned with all the available keys. Then you can change some values if you want to adjust some text. If you need a new language just get the original JSON file, change it and save it for your language

      If you remove some keys from the JSON file, the defaults from the English version will be used.

      In the preview mode the JSON is merged with our default translations preferring your custom ones. In the editing mode the missing fields won't be shown - only those that you explicitly set will. If you notice untranslated keys in the preview mode, add those into the JSON in the editing mode.
